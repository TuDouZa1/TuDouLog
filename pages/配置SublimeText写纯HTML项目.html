安装插件

Win/Linux: ctrl+shift+p, Mac: cmd+shift+p输入

Install Package Control

回车即可

一些推荐插件

  • ChineseLocalizations
  • Emmet
  • AutoFileName：在写 src=", href=" 时，自动提示并补全当前目录下的文件名。
  • CSS3：CSS语法高亮与补全
  • HTML-CSS-JS Prettify：代码格式化
  • Terminus：内部集成终端
  • BracketHighlighter：括号高亮增强
  • ColorHelper：颜色选择器

推荐配置JSON

{
    // 保存时自动格式化（需先安装 HTML-CSS-JS Prettify）
    "format_on_save": true,
    // 显示相对行号，便于快速跳转
    "line_numbers": true,
    "relative_line_numbers": true,
    // 高亮当前行
    "highlight_line": true,
    // 平滑滚动
    "scroll_speed": 1.0,
    "smooth_scrolling": true
}

卸载插件

Win/Linux: ctrl+shift+p, Mac: cmd+shift+p输入

Remove Package

使用键盘上下键选择你想要卸载的插件（例如 Terminus），然后按 Enter 键。

也可以打开插件文件夹手动删除

本地服务器运行

npx serve // 需要nodejs
python -m http.server // 需要python
